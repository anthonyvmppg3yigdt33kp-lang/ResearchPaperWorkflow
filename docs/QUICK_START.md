# Quick Start

The maintained V4.3 quick-start path is natural-language first:

1. Open Codex or Claude in this repository.
2. Describe your research situation, target journal, available materials, and
   the next workflow boundary you want to reach.
3. Ask the model to advance one safe stage and stop at checkpoints, missing
   inputs, pending harness work, gate failures, or stale artifacts.
4. Review the reported `paper_id`, stage truth, missing artifacts, and required
   human decisions.

Use these current guides:

- `README.md`
- `USER_GUIDE.md`
- `ARCHITECTURE.md`
- `docs/OPERATION_GUIDE_ZH.md`
- `docs/NEXT_GEN_V4_TRUTH_LAYER.md`

Older pre-truth-layer quick-start instructions have been retired from this file.
