# Multi-omics Evidence Synthesis Planner

This module is a planning and audit contract for linking modality-specific results to claim boundaries. Researchers can inspect `config/bioinformatics_method_contract.yaml` and the generated method-source catalog to see required inputs: modality-specific result tables, sample matching tables, source maps, statistical units, and reviewer-risk annotations.

Use this asset to decide whether evidence supports discovery, association, validation, or figure-level claims. It does not execute an omics model by itself and must not be used to upgrade weak or unmatched evidence into mechanistic conclusions.
