# Python Built-in Bulk RNA-seq Pilot

This module is a standalone Python method asset for lightweight execution testing. Researchers can open `main.py` to inspect argument parsing, count/metadata loading, logCPM calculation, approximate group contrast, Benjamini-Hochberg FDR, sample QC export, SVG volcano/heatmap previews, and source-map generation.

Use this asset to verify that the graph executor can run Python method assets and write scoped outputs. It is not a publication-grade DESeq2, edgeR, or limma replacement, and manuscript claims must use a validated statistical environment and biological replicate design.
